Plan B - Vue Front-only (API wired)
-----------------------------------
1) Copy your card PNGs (A_H.png ... K_S.png, BACK.png) into: public/cards/
2) Copy chip_100/500/1000/5000.png into: public/assets/
3) Run:
   npm i
   npm run dev
4) Backend base is http://localhost:8080 (override with VITE_API_BASE)
